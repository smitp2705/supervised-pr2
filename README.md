#  Supervised Learning - Ridge & Lasso Regression

A machine learning project demonstrating the implementation of **Ridge Regression** and **Lasso Regression** for predicting house prices using a real-world housing dataset. This project covers the complete machine learning workflow, including data preprocessing, feature engineering, model training, hyperparameter tuning, and performance evaluation.

---

##  Project Overview

The objective of this project is to build regression models that accurately predict house prices while minimizing overfitting using regularization techniques.

The notebook includes:

- Data Loading
- Data Cleaning
- Missing Value Analysis
- Duplicate Record Checking
- Date Feature Engineering
- Feature Scaling
- Train-Test Split
- Ridge Regression
- Lasso Regression
- Hyperparameter Tuning using GridSearchCV
- Model Evaluation
- Performance Comparison

---

##  Dataset

The dataset contains housing-related information such as:

- House Features
- Sale Date
- Property Details
- Location Information
- Price of House (Target Variable)

**Target Variable**

- `house_price_inr`
